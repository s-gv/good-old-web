Orange Forum

Orange Forum is an easy to deploy forum that has few dependencies. It is written in golang and a compiled binary is available for linux. Try the live demo hosted here.

By default, sqlite is used, so it's easy to get started. Download the binary and migrate the database with: 

./orangeforum -migrate

Create a superadmin: 

./orangeforum -createsuperuser

Finally, start the server: 

./orangeforum

More details about configuration and command line options are here.
