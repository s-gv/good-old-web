Good Old Web

Good Old Web is a collection of open-source, slim webapps that make very little use of javascript and Web 2.0 features like AJAX. The rendered pages are light-weight and work even if javascript is disabled. This is an attempt to combat the website obesity crisis and reduce web bloat. Please contact info@goodoldweb.com if you have any questions or want support.

Orange Forum

Orange Forum is an easy to deploy forum that has few dependencies. It is written is golang and a compiled binary is available for linux. Try the latest version hosted here.

By default, sqlite is used, so it's easy to get started. Download the binary and migrate the database with: 

./orangeforum -migrate

Create a superadmin with: 

./orangeforum -createsuperuser

Now, start the server: 

./orangeforum

More details about configuration and command line options are here.
