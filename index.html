Good Old Web

Good Old Web is a collection of open-source, slim webapps that make very little use of javascript and Web 2.0 features like AJAX. The rendered pages are light-weight and work even if javascript is disabled. This is an attempt to combat the website obesity crisis and reduce web bloat. If you have any questions, send an e-mail to: info@goodoldweb.com

Orange Forum

Orange Forum is an easy to deploy forum that has few dependencies. It is written is golang and a compiled binary is available for linux. Try the latest version here.

By default, sqlite is used, so it's easy to get started. Download the binary and create an admin user with: 

./orangeforum --migrate && ./orangeforum --createsuperuser

Now, start the server:

./orangeforum
